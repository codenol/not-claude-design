# Skala DS

Design System skeleton — parity environment: React ↔ Figma.

**Stack:** React + TypeScript + Vite / Figma MCP / CSS Modules + tokens

## Structure

| Layer | Path |
|-------|------|
| Directives | `directives/` |
| Tokens | `artifacts/tokens.json` → `app/src/tokens/` |
| Components | `app/src/components/{atoms,molecules,organisms}/` |
| Figma config | `figma.config.json` |

## Quick start

```bash
cd app && npm install && npm run dev
```
