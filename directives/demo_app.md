# Цель: собрать демо-страницу из существующих компонентов.

## Layout

Демо-страница использует компонент `Layout` (organism), который композитит:

```
┌──────────────────────────────────────────────────────────────┐
│ Layout (full viewport)                                       │
│ ┌──────────────────────────────────────────────────────────┐│
│ │ Sidebar    │ Content (flex: 1)                            ││
│ │ 249px/97px │ ┌──────────────────────────────────────────┐ ││
│ │            │ │ Nav — Breadcrumbs (48px)                 │ ││
│ │            │ ├──────────────────────────────────────────┤ ││
│ │            │ │ Main container (flex: 1, scrollable)     │ ││
│ │            │ │ ┌──────────────────────────────────────┐ │ ││
│ │            │ │ │ Все компоненты ДС (section)          │ │ ││
│ │            │ │ └──────────────────────────────────────┘ │ ││
│ │            │ └──────────────────────────────────────────┘ ││
│ └────────────┴──────────────────────────────────────────────┘│
└──────────────────────────────────────────────────────────────┘
```

### Структура Layout

| Зона | Компонент | Размеры |
|------|-----------|---------|
| Sidebar | `Sidebar` (organism) | 249px expanded / 97px collapsed |
| Nav | `Breadcrumbs` (molecule) | height 48px |
| Main container | scrollable область | flex: 1, overflow-y: auto |
| Gap Sidebar↔Content | — | 16px |
| Padding Layout | — | 20px |

### Где

- `app/src/components/organisms/Layout/` — компонент Layout
- `app/src/App.tsx` — точка входа, оборачивает все секции в `<Layout>`
- Все компоненты ДС размещаются внутри `<Layout>` как children

## Подключение

- Добавить новые компоненты внутрь `<Layout>` в `App.tsx`
- Каждый компонент — отдельная `<section>` с `<h2>` заголовком
- Breadcrumbs по умолчанию: `Главная / Компоненты`
